# COMANDOS SQL ESTUDADOS

## DML (Data Manipulation Language)

| Comando | Descrição |
|---------|-------------|
| **SELECT** | Recuperar dados do banco de dados |
| **INSERT** | Inserção de dados em uma tabela |
| **UPDATE** | Atualização de dados existentes em uma tabela |
| **DELETE** | Exclusão de registros de uma tabela do banco de dados |

## DDL (Data Definition Language)

| Comando | Descrição |
|---------|-------------|
| **CREATE TABLE** | Usado para criar uma tabela no banco de dados |
| **DROP TABLE** | Exclusão de uma tabela no banco de dados |

# CLÁUSULAS ESTUDADAS

| Cláusula | Descrição |
|----------|-------------|
| **FROM** | Especifica a tabela de origem dos dados |
| **WHERE** | Filtra registros com base em condições (AND, OR, NOT) |
| **JOIN** | Combina linhas de duas ou mais tabelas com base em uma coluna relacionada |

### Tipos de JOIN:
- `INNER JOIN` - Retorna apenas registros que têm correspondência em ambas as tabelas
- `LEFT JOIN` - Retorna todos os registros da tabela esquerda e os correspondentes da direita
- `RIGHT JOIN` - Retorna todos os registros da tabela direita e os correspondentes da esquerda
